Frame Blocking

This demo explores how to employ frame-blocking to prevent a Web page from appearing within a frame. Click the link in the left frame to view the file imusic.htm. The page will appear within the right frame. However, if you add the following JavaScript commands to imusic.htm

if (top.location.href != self.location.href) { 

   top.location.href = self.location.href;
}

and run them when the page is loaded the browser will compare the URL of the topmost window to the URL of imusic.htm. If different, JavaScript will change the location of the topmost window to the location of the current window. As a result, imusic.htm will jump to the top of the frame hierachy, replacing the entire frameset.
