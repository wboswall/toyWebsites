Creating Custom Sliders

You can use JavaScript to create your own custom slider objects. All of the sliders on this page have been created at WidgetMage using JavaScript custom objects. Click and drag the slider bars to change the color, opacity, and z-index of the sample boxes. You can learn how to create your own sliders by visiting our Slider Tutorial page and our white paper on Custom Objects.
