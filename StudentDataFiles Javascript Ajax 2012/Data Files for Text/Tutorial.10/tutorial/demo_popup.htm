Pop-up Windows Features

This demo displays how to control the appearance of a pop-up window with JavaScript. Enter the window's dimensions and select the features that you want to include in the window, and then click the Show pop-up window button. The JavaScript code to create the pop-up window is shown in the box at the bottom of the document.
