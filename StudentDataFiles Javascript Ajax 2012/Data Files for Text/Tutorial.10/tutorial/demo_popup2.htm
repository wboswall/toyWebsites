Pop-up Windows Methods

This demo displays how to use methods of the window object to control the position and size of a pop-up window. To display the pop-up window, click the Show Pop-up Window button. To move the window, click the moving buttons. To resize the window, click the resizing buttons. To close the pop-up window, click the Close Pop-up Window button. Each time you click one of the control buttons, the JavaScript command to manipulate the window appears in the box at the bottom of the page.
